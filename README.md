# 📘 Fiap-Estudos

**Aqui eu vou postar todos os dias as aulas que eu tive na Fiap cursando Engenharia de Software.**

Este repositório serve como um diário de estudos, contendo registros organizados de todas as aulas que eu participei durante o curso. O conteúdo é separado por dias e disciplinas, facilitando a consulta e a revisão sempre que necessário.

## 📅 Organização

Os conteúdos estão organizados da seguinte forma:

Fiap-Estudos/
├── Matéria/
│ ├── Data/
│ ├── Aula/
│ ├── ...
├── Matéria/
│ ├── ...

Dentro de cada pasta das matérias, você encontrará os arquivos correspondentes a cada dia de aula, com o nome no formato:
Materia - Data  - .md


Exemplo:
Python - 2025-03-04 - Algoritmos.md


## 🧾 Conteúdos Incluídos

- Resumos das aulas
- Anotações pessoais
- Códigos e exercícios práticos
- Tópicos abordados em sala
- Observações importantes passadas pelos professores
  **Nem sempre pode ser assim**

## ✍️ Atualizações Diárias

O repositório será atualizado diariamente conforme o andamento das aulas, servindo tanto como histórico quanto como material de apoio para revisões e estudos futuros.

## 📌 Observação

Este é um repositório pessoal de estudos. Os conteúdos refletem meu entendimento das aulas e podem conter observações individuais para facilitar meu aprendizado.
